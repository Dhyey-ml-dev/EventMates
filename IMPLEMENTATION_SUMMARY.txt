Status:          ✅ WORKING

Login Flow:
  1. User enters credentials at /admin/login
  2. POST /api/admin/login validates
  3. Backend checks:
     ├─ Email & password required
     ├─ User exists & is admin role
     ├─ Account is verified
     ├─ Account is not blocked
     └─ Password hash matches
  4. JWT tokens generated
  5. Tokens stored in localStorage
  6. Redirect to /admin/dashboard
  7. All admin routes now accessible

   - /api/admin/login (public)
   - /api/admin/* (protected)

✅ Authentication
   - Email & password validation
   - Bcrypt password hashing
   - JWT token generation
   - Token expiration
   - Refresh token rotation

✅ Authorization
   - protect() middleware - JWT verification
   - authorize('admin') middleware - Role check
   - Only admins can access endpoints

   - IP address tracking

✅ Activity Logging
