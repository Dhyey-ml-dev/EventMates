### ✅ PART 3: ORGANIZER PAID EVENT POSTING (Complete Monetization)

**Payment Flow:**

```
Organizer → Event Form → Payment Page → Razorpay → Event Published
```

**Implementation:**

**A. Event Creation (Draft)**
- Event saved with status: "draft"
- Event saved with paymentStatus: "pending"
- Event NOT visible to public yet

**B. Payment Selection**
- Basic Plan: ₹299 (30-day visibility)
- Featured Plan: ₹599 (30 days + ⭐ badge)

**C. Razorpay Checkout**
- User enters card details securely
- Payment processed by Razorpay
- Signature verified by backend

**D. Event Publishing**
- After successful payment:
  - Event status: "draft" → "published"
  - paymentStatus: "pending" → "paid"
  - If featured: isFeatured: true
- Event now visible to all users

**E. Revenue:**
- ₹299-₹599 per event per month
- Scalable to multiple organizers
